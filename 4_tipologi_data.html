Data Set Information:

This database contains 76 attributes, but all published experiments refer to using a subset of 14 of them. In particular, the Cleveland database is the only one that has been used by ML researchers to this date. The "goal" field refers to the presence of heart disease in the patient. It is integer valued from 0 (no presence) to 4. Experiments with the Cleveland database have concentrated on simply attempting to distinguish presence (values 1,2,3,4) from absence (value 0).

The names and social security numbers of the patients were recently removed from the database, replaced with dummy values.

One file has been "processed", that one containing the Cleveland database. All four unprocessed files also exist in this directory.

To see Test Costs (donated by Peter Turney), please see the folder "Costs"

Complete attribute documentation:

  1. age: age in years
  2. sex: sex (1 = male; 0 = female)
  3. cp: chest pain type
    • Value 1: typical angina
    • Value 2: atypical angina
    • Value 3: non-anginal pain
    • Value 4: asymptomatic
  4. trestbps: resting blood pressure (in mm Hg on admission to the hospital)
  5. chol: serum cholestoral in mg/dl
  6. fbs: (fasting blood sugar > 120 mg/dl) (1 = true; 0 = false)
  7. restecg: resting electrocardiographic results
    • Value 0: normal
    • Value 1: having ST-T wave abnormality (T wave inversions and/or ST elevation or depression of > 0.05 mV)
    • Value 2: showing probable or definite left ventricular hypertrophy by Estes' criteria
  8. thalach: maximum heart rate achieved
  9. exang: exercise induced angina (1 = yes; 0 = no)
  10. oldpeak = ST depression induced by exercise relative to rest
  11. slope: the slope of the peak exercise ST segment
    • Value 1: upsloping
    • Value 2: flat
    • Value 3: downsloping
  12. ca: number of major vessels (0-3) colored by flourosopy
  13. thal: 3 = normal; 6 = fixed defect; 7 = reversable defect
  14. num: diagnosis of heart disease (angiographic disease status)
    • Value 0: < 50% diameter narrowing
    • Value 1: > 50% diameter narrowing (in any major vessel: attributes 59 through 68 are vessels)

Data Kategori¶

Data kategoris adalah kumpulan informasi yang dibagi menjadi beberapa kelompok. Data ini disebut kategorikal karena dapat dikelompokkan sesuai dengan variabel yang ada dalam biodata seperti jenis kelamin, negara tempat tinggal, dll.

Data kategoris dapat menggunakan nilai numerik (seperti "1" menunjukkan Ya dan "2" menunjukkan Tidak), tetapi angka tersebut tidak memiliki arti matematis. Seseorang tidak dapat menjumlahkan atau menguranginya satu sama lain.

Data kategorik terdiri dari dua kategori, yaitu; data nominal, data ordinal, dan interval. Data nominal atau disebut juga data bernama adalah jenis data yang digunakan untuk menamai variabel, sedangkan data ordinal adalah jenis data dengan skala atau urutannya.

Data nominal:

Ini adalah jenis data yang digunakan untuk memberi nama variabel tanpa memberikan nilai numerik apa pun. Diciptakan dari nomenklatur Latin Nomen (artinya nama), tipe data ini adalah subkategori data kategorikal. Data nominal terkadang disebut data "berlabel" atau "bernama". Contoh data nominal termasuk nama, warna rambut, jenis kelamin dll.

Tipe data ini bersifat deskriptif, karena terkadang memungkinkan responden kebebasan untuk mengetik jawaban. Meskipun karakteristik ini membantu dalam mencapai kesimpulan yang lebih baik, terkadang hal ini menimbulkan masalah bagi peneliti karena mereka harus berurusan dengan begitu banyak data yang tidak relevan.

Data Nominal¶

Data chestpain (cp) merupakan data kategori tipe nominal dimana cp cp dengan nilai:

  • 1: typical angina
  • 2: atypical angina
  • 3: non-anginal pain
  • 4: asymptomatic

Distribusi data berdasarkan kelompok kategori chestpain ditampilkan pada grafik di bawah ini

Data Nominal - Dikotomi¶

Data jumlah orang berdasar sex merupakan data nominal yang dibedakan berdasarkan perbedaan jenis kelamin: laki-laki (1) atau perempuan (0). Distribusi frekuensi data berdasar jenis kelamin digambarkan pada grafik di bawah ini.

Data jumlah orang dengan exang/ exercise induced angina terdiri dari dua kategori nominal dikotomi, nilai 1 = iya, nilai 0 = tidak.

fbs merupakan data gula ketika puasa (fasting blood sugar > 120 mg/dl). Nilai fbs 1 = true; 0 = false.

Data Ordinal¶

Ini adalah tipe data dengan urutan atau skala yang ditetapkan. Namun, urutan ini tidak memiliki skala standar untuk mengukur perbedaan variabel di setiap skala. data nominal

restecg adalah hasil resting electrocardiographic, dengan nilai:

  • 0: normal
  • 1: having ST-T wave abnormality (T wave inversions and/or ST elevation or depression of > 0.05 mV)
  • 2: showing probable or definite left ventricular hypertrophy by Estes' criteria

Data slope menggambarkan slope puncak olah raga, nilai:

  • 1: upsloping
  • 2: flat
  • 3: downsloping

Data Interval

Skala interval pengukuran data adalah skala di mana tingkatan-tingkatannya diurutkan dan setiap jarak yang sama secara numerik pada skala memiliki perbedaan interval yang sama. Jika itu merupakan perpanjangan dari skala ordinal, dengan perbedaan utamanya adalah keberadaan interval yang sama.

fbs: (fasting blood sugar > 120 mg/dl) (1 = true; 0 = false)

Data kategoris bersifat kualitatif. Artinya, ini mendeskripsikan peristiwa menggunakan string kata daripada angka.

Dalam kasus data ordinal, yang memiliki urutan atau skala tertentu, skala tersebut tidak memiliki interval standar. Ini tidak berlaku untuk data nominal.

Meskipun data kategoris bersifat kualitatif, terkadang data tersebut memerlukan nilai numerik. Akan tetapi, nilai-nilai ini tidak menunjukkan karakteristik kuantitatif. Operasi aritmatika tidak dapat dilakukan padanya.

Data kategorikal juga dapat diklasifikasikan menjadi biner dan non biner tergantung pada sifatnya. Pertanyaan yang diberikan dengan opsi Ya atau Tidak diklasifikasikan sebagai biner karena memiliki dua opsi sementara menambahkan Mungkin ke opsi yang diberikan akan menjadikannya non biner.

Data Kontinu¶

Data kontinu merupakan data kuantitatif yang dapat diukur. Data ini memiliki jumlah kemungkinan nilai tak terbatas dalam rentang yang dipilih, mis. kisaran suhu, kadar air, dll.

Histogram di bawah ini merupakan nilai sebaran frekuensi kadar kholeterol.

Penggambaran kadar kholesterol dengan bentuk kurva density lebih mudah untuk diamati, sebagaimana gambar di bawah ini.

Sebaran usia pasien juga merupakan data kontinyu yang dikelompokan dalam rentang tahun. Kurva distribusi untuk usia pasien ditampilkan dalan Gambar di bawah ini.
